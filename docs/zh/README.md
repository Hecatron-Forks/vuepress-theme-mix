### 🚀 使用模板

```sh:no-line-numbers
// 克隆模板仓库到本地的 mydocs 文件夹下
git clone https://github.com/gavinliu6/vuepress-theme-mix-template.git mydocs

// 切换工作目录
cd mydocs

// 安装依赖
yarn

// 启动开发服务器
yarn docs:dev
```

在浏览器中打开 [http://127.0.0.1:8080](http://127.0.0.1:8080) 即可预览项目。:tada:
